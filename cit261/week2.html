Reading Notes

Chapter 2: Programming Basics

Variables

  • var is older version of declaring variables
  • use let instead of var

Scope

  • Global Scope: accessible everywhere in the program. Not considered good practice.
  • Local Scope: any variables defined inside a block are only available inside that block.

Converting/Parsing

  • Strings to Numbers: Number('str');
  • Numbers to Strings: String(num);
  • Parsing Numbers: parseInt() or parseFloat()

Template Literals

  • Use the backtick '`' to deliminate a string (can use any combo of quotes within the string).

Chapter 3: Arrays, Logic, and Loops

Array Methods

  • array.length; - find the length of an array
  • pop() - remove the last item from an array
  • shift() - removes the first item in an array
  • push() - appends an new value to the end of the array
  • unshift() - appends a new value to the beginning of an array
  • concat() - merge an array with one or more arrays
  • spread ... - spread the elements of two arrays and put them in a new array
  • join() - can be used to turn the array into a string. Default is separated with commas
  • slice() - chops out a slice of an original array, start and end index
  • splice() - removes item from an array then inserts new items.
  • reverse() - reverse the order of an array
  • sort() - sort the order of an array
  • indexof() - returns the index of an item
  • includes() - returns a boolean if array contains an element

Chapter 4: Functions

Ways to declare a function

  • function functionName(){
  • const variable = function(){
  • const variable = new Function('statment;');
  • Arrow Functions: const square = x => x*x;

Callbacks

  • A function that is passed as an argument to another.
  • Example: function functionName(x, callback)
